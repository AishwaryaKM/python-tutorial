Getting Started

The box below provides an area where you can write python code. You can then execute the python code using the Execute button below it. The output from the python program you wrote will be displayed above it.

An example program is provided which you can load using the Fill in example button. Don't worry about understanding the program at this stage, just execute it for now.

To learn python it is important to write python code. This tutorial will show you plenty of code snippets but please do type them out yourself to train your fingers.
