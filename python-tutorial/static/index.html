Online Python Tutorial

This is an experimental project aiming to provide an interactive python interpreter to casual web visitors so that they can familiarise themselves with the python programming language. It is not ready for general use so I recommend visiting the officlal python tutorial instead, for now. The source code for the project can be found on github.com.

The Online Python Tutorial is hosted
	on Google App
	Engine [GAE] which provides free python web hosting on the
	google infrastructure.  Users of this site can submit python
	code from their browser to run on the google servers.  The
	submitted code runs in a restricted execution environment to
	protect the application and to protect users from each other.
	At the moment submitted code must conform to the rules imposed
	by
	CapPython but
	this restriction may be relaxed in the future.  No standard
	library modules are available.  CapPython is written in pure
	python but relies on the built-in python parser
	module which is not available under GAE.  To work around this
	the Online Python Tutorial uses a mostly complete Python 2.5
	parser in pure-python taken from
	the PyPy project.
