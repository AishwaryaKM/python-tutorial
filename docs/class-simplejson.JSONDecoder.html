JSONDecoder

Simple JSON <http://json.org> decoder

The JSONDecoder class is accessible via the simplejson module.

  • Attributes (1)
  • Methods (1)
  • Source

Performs the following translations in decoding:

JSON Python
object dict
array list
string unicode
number (int) int, long
number (real) float
true True
false False
null None

It also understands NaN, Infinity, and -Infinity as their corresponding float values, which is outside the JSON spec.
